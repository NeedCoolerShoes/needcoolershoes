## Run Locally

Clone the project

```bash
git clone https://github.com/needcoolershoes/needcoolershoes
```

Install PostgreSQL

```bash
sudo apt install postgresql
```

Enter the repository folder

```bash
cd needcoolershoes
```

Install `foreman`

```bash
gem install foreman
```

Start the webserver

```bash
foreman start
```
